# 2D-Sidescroller-Project
2D Sidescroller Game in C++ using Allegro

##Authors: Richard Diaz / Brandon Martin

### Controls:
* Start - Enter
* Pause/Menu - Spacebar
* Movement - Arrow Keys
* Slide - Down + Direction
* Select weapon in Menu - Mouse Click
* Use weapon - Q
